# Sync-Web 🚀
### Multi-Threaded Real-Time OS Resource & Concurrency Simulator

Sync-Web is a comprehensive, full-stack simulation dashboard built to visually demonstrate high-level Operating System abstractions in real-time. 

## 🛠️ Core Features Simulated
* **IT Support Dept:** CPU Scheduling via Non-Preemptive **Shortest Job First (SJF)**.
* **Admin Dept:** Dynamic Multi-Level Feedback Queue (**MLFQ**) with aging simulation.
* **HR Dept:** Memory allocation contrast showing **Contiguous Allocation** (External Fragmentation) vs. **Paged Allocation**.
* **Finance Dept:** Demonstrating Concurrency control using **Unmanaged Race Conditions** vs. **Mutex Locks**.

## 💻 Tech Stack
* **Backend:** Python, Flask, Native Threading Architecture
* **Real-Time Communication:** Flask-SocketIO (WebSockets)
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Dynamic DOM Updates)
